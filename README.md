# Automated Email Sender using Python

## Screenshots

##  Project Overview
This project is a Python-based Automated Email Sender that uses the SMTP protocol to send emails automatically to multiple recipients.

## Features
- Bulk email sending
- Gmail SMTP integration
- Secure authentication using App Password
- Dynamic recipient management using text files
- Custom email message handling

##  Technologies Used
- Python
- SMTP
- Gmail App Password

##  Project Structure
- main.py → Starts the application
- email_sender.py → Core email logic
- recipients.txt → List of recipient emails
- message.txt → Email content
